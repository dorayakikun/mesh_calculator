Getting Started

Waffle Map はメッシュコードからメッシュを描画する地図です。

機能

Waffle Map は以下の機能を提供します。

  • メッシュコードからメッシュを描画
  • マーカーの配置
  • タイルグリッドの表示
  • メッシュグリッドの表示

インストール

npm を利用して Waffle Map をインストールしてください

npm i
npm run dev
